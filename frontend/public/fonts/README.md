# Fonts Directory

Place your `.woff2` variable font files here.

- `GeneralSans-Variable.woff2`: Download from [Fontshare](https://www.fontshare.com/fonts/general-sans).
- `JetBrainsMono-Variable.woff2`: Download from [JetBrains](https://www.jetbrains.com/lp/mono/).

Due to air-gapping requirements, fonts are hosted locally. Empty placeholders are provided to satisfy compiler checks initially, but they must be replaced by actual font binaries.
