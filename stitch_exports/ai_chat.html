Certainly. A synapse in a computational neural network is often represented as a weight connecting two nodes. Below is a simplified Python model using NumPy to simulate signal transmission across a synaptic cleft.

synapse_sim.py Python 
import numpy as np

class Synapse:
    def __init__(self, weight=0.5):
        self.weight = weight

    def transmit(self, signal):
        # Apply synaptic weight and non-linearity
        return np.tanh(signal * self.weight)

# Initialize simulation
syn = Synapse(weight=0.85)
output = syn.transmit(0.5)
